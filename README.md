# 🚀 Portfolio Full-Stack – Alvaro Marchan
Portfolio personal full-stack desarrollado con tecnologías modernas, diseñado para mostrar mis habilidades como Full-Stack Developer en desarrollo web, APIs y gestión de bases de datos.

## 🧑‍💻 Sobre el proyecto
Este proyecto es una aplicación web que incluye:

* __Frontend__ moderno y responsive desarrollado con Angular.
* __Backend__ basado en Spring Boot con arquitectura REST.
* __Base de datos__ SQL Server para almacenamiento de información.
* Estructura modular y escalable siguiendo buenas prácticas profesionales.

El objetivo es presentar mis proyectos, tecnologías y experiencia de forma clara, visual y funcional.

## 🛠️ Tecnologías utilizadas

* __Frontend__
    * Angular
    * TypeScript
    * PrimeNG
    * TailwindCSS

* __Backend__
    * Java
    * Spring Boot
    * Spring Data JPA
    * REST APIs

* __Base de Datos__
    * SQL Server

* __Otros__
    * Git y GitHub
    * Git Flow (feature branches, develop, main)
    * Postman (testing APIs)
